git clone git@github.com:Sampson-Huang/raspberry.git
将GitHub仓库克隆到本地，最后为仓库的ssh地址，上面这个是自己这个仓库的地址

git add <文件名>
将文件添加到暂存区，可同时添加多个文件

git commit -m"说明文字"
提交修改之前添加的文件

git push origin master
推送到远程仓库
